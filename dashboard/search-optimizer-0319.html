数据接入说明

Gclid 检测：使用 Python 拨测脚本（requests/playwright），定时请求各 Final URL 并追加 ?gclid=test_xxx 参数，检查经 301/302 跳转后 gclid 是否保留在最终 URL 中。

LCP 检测：接入 Google PageSpeed Insights API（https://www.googleapis.com/pagespeedonline/v5/runPagespeed?url=URL&strategy=mobile），提取 lighthouseResult.audits['largest-contentful-paint'].numericValue（毫秒），每日定时跑一次。

当前为 Mock 数据演示。后续替换为真实拨测结果后，将 data-tag-mock 改为 data-tag-real。
